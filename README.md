# Stackcraftio's Calculator 

This is my first web application built using **Python and Flask**.  
It’s a simple calculator with a pastel-themed UI and basic functionality, created as part of my learning journey in software development.

##  Features
- Clean pastel UI with glass-style design  
- Basic arithmetic operations (+, −, ×, ÷)  
- Percentage (%) calculations  
- Keyboard input support  
- Python Flask backend  

##  How to Run

Clone the repository:
```bash
git clone https://github.com/stackcraftio/Calculator.git
cd Calculator

pip install flask

python app.py

http://127.0.0.1:5000
